1. Create a database ‘assignment_db’
2. Import the attached sql file (3 tables) in the same database. These tables represent physical equipment (table number).
3. Place the project folder in htdocs folder and start the xampp server.
4. Please configure the DBController with user and password of database.
5. Open calculate.php page in browser to generate the tables as mentioned in the attached excel.
6. Navigate to ouput folder in project root ddirectory to see the generated tables.
7. Open index.php page in browser to see data from the new table.
8. Select equipment (table number) and periodicity (daily/weekly) from dropdown to see results of respective.
9. The web page will show data on chart only for periodicity (daily/weekly).
